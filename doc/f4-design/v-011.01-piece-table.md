# v-011.01 — Piece Table Buffer

Parent: [v-011-kairn-spec.md](v-011-kairn-spec.md)

## Overview

The piece table is kairn's core text buffer. It replaces the `Vec<String>`
line-array model from feature/mini-ide with a structure that provides:
- O(log n) insert and delete at any position
- Efficient undo/redo via history snapshots
- Minimal memory copying (original text is never modified)
- Natural mapping to LSP incremental document sync

## Data model

```rust
/// Which buffer a piece references.
#[derive(Clone, Copy, PartialEq, Eq, Debug)]
enum Source {
    /// The original file content (immutable after load).
    Original,
    /// The append-only buffer for inserted text.
    Add,
}

/// A contiguous run of text from one source buffer.
#[derive(Clone, Debug)]
struct Piece {
    source: Source,
    start: usize,   // byte offset into the source buffer
    len: usize,     // byte length
}

/// The piece table buffer.
pub struct PieceTable {
    /// Original file content. Never modified after creation.
    original: String,
    /// Append-only buffer for all inserted text.
    add_buf: String,
    /// Ordered list of pieces describing the document.
    pieces: Vec<Piece>,
    /// Cached total byte length.
    total_len: usize,
    /// Line index for fast line ↔ offset mapping.
    line_index: LineIndex,
    /// Undo/redo history.
    history: UndoHistory,
    /// Whether the buffer has been modified since last save.
    modified: bool,
    /// File path (if loaded from file).
    file_path: Option<String>,
}
```

## Public API

```rust
impl PieceTable {
    // ── Construction ──

    /// Create an empty buffer.
    pub fn new() -> Self;

    /// Create a buffer from file content.
    pub fn from_str(content: &str) -> Self;

    /// Load from a file path. Returns error if file can't be read.
    pub fn from_file(path: &str) -> Result<Self>;

    // ── Read ──

    /// Total byte length of the document.
    pub fn len(&self) -> usize;

    /// Whether the document is empty.
    pub fn is_empty(&self) -> bool;

    /// Total number of lines (always >= 1).
    pub fn line_count(&self) -> usize;

    /// Get the content of a single line (0-indexed). No trailing newline.
    pub fn line(&self, line_num: usize) -> Option<String>;

    /// Get a range of lines as a single string.
    pub fn lines_range(&self, start: usize, end: usize) -> String;

    /// Get the full document content as a string.
    pub fn content(&self) -> String;

    /// Get a byte range of the document.
    pub fn slice(&self, byte_start: usize, byte_end: usize) -> String;

    /// Convert (line, col) to byte offset. Col is in characters, not bytes.
    pub fn line_col_to_offset(&self, line: usize, col: usize) -> Option<usize>;

    /// Convert byte offset to (line, col).
    pub fn offset_to_line_col(&self, offset: usize) -> (usize, usize);

    // ── Write ──

    /// Insert text at a byte offset.
    pub fn insert(&mut self, offset: usize, text: &str);

    /// Delete a byte range.
    pub fn delete(&mut self, start: usize, end: usize);

    /// Replace a byte range with new text.
    pub fn replace(&mut self, start: usize, end: usize, text: &str);

    /// Insert text at (line, col).
    pub fn insert_at(&mut self, line: usize, col: usize, text: &str);

    /// Delete from (line1, col1) to (line2, col2).
    pub fn delete_range(
        &mut self,
        line1: usize, col1: usize,
        line2: usize, col2: usize,
    );

    // ── Undo/Redo ──

    /// Undo the last edit. Returns false if nothing to undo.
    pub fn undo(&mut self) -> bool;

    /// Redo the last undone edit. Returns false if nothing to redo.
    pub fn redo(&mut self) -> bool;

    /// Start a group — all edits until end_group are undone as one unit.
    pub fn begin_group(&mut self);

    /// End the current group.
    pub fn end_group(&mut self);

    /// Number of undoable operations.
    pub fn undo_count(&self) -> usize;

    /// Number of redoable operations.
    pub fn redo_count(&self) -> usize;

    // ── State ──

    /// Whether the buffer has been modified since last save/load.
    pub fn is_modified(&self) -> bool;

    /// Mark the buffer as unmodified (called after save).
    pub fn mark_saved(&mut self);

    /// The file path, if any.
    pub fn file_path(&self) -> Option<&str>;

    /// Set the file path.
    pub fn set_file_path(&mut self, path: &str);

    // ── LSP sync ──

    /// Get the incremental change since the last call to `take_changes()`.
    /// Returns a list of (range, new_text) suitable for LSP didChange.
    pub fn take_changes(&mut self) -> Vec<TextChange>;
}

/// A text change for LSP incremental sync.
pub struct TextChange {
    pub start_line: usize,
    pub start_col: usize,
    pub end_line: usize,
    pub end_col: usize,
    pub new_text: String,
}
```

## Internal operations

### Insert

1. Find which piece contains `offset` (binary search on cumulative lengths)
2. Split that piece at the offset point → two pieces
3. Append `text` to `add_buf`, record its start position
4. Insert a new piece (Source::Add, start, len) between the two halves
5. Update `total_len`
6. Update `line_index` incrementally
7. Record change for undo and LSP sync

### Delete

1. Find the pieces spanning `[start, end)`
2. Split the first piece at `start` if needed
3. Split the last piece at `end` if needed
4. Remove all pieces fully within the range
5. Update `total_len`
6. Update `line_index` incrementally
7. Record change for undo and LSP sync

### Replace

Equivalent to delete + insert, but recorded as a single undo operation.

## Line index

```rust
struct LineIndex {
    /// Byte offset of the start of each line. line_starts[0] is always 0.
    line_starts: Vec<usize>,
}

impl LineIndex {
    /// Build from full document content.
    fn build(content: &str) -> Self;

    /// Rebuild after an edit. Only recomputes the affected region.
    fn update_insert(&mut self, offset: usize, text: &str);
    fn update_delete(&mut self, start: usize, end: usize);

    /// Line count.
    fn line_count(&self) -> usize;

    /// Byte offset of line start.
    fn line_start(&self, line: usize) -> Option<usize>;

    /// Which line contains this byte offset.
    fn offset_to_line(&self, offset: usize) -> usize;
}
```

The line index is rebuilt incrementally on each edit:
- Insert: scan the inserted text for `\n`, insert new entries into `line_starts`
- Delete: remove entries in the deleted range, adjust offsets after

For the initial implementation, a `Vec<usize>` with binary search is
sufficient. If profiling shows it's a bottleneck for very large files
(100K+ lines), replace with a B-tree.

## Undo/redo

```rust
struct UndoHistory {
    undo_stack: Vec<EditRecord>,
    redo_stack: Vec<EditRecord>,
    /// Active group (if begin_group was called).
    group: Option<Vec<EditRecord>>,
}

/// A single edit that can be undone.
struct EditRecord {
    /// The pieces list before this edit.
    pieces_snapshot: Vec<Piece>,
    /// The line index before this edit.
    line_starts_snapshot: Vec<usize>,
    /// Cursor position before this edit (stored by editor, not buffer).
    cursor_before: Option<(usize, usize)>,
}
```

Undo: pop from undo_stack, push current state to redo_stack, restore
the snapshot. Redo: reverse.

Grouped edits (e.g., `dd` = delete line) are collected into a single
`EditRecord` containing the state before the first edit in the group.

### Memory optimization

Storing full piece list snapshots is simple but memory-heavy for large
files with many edits. For the initial implementation, this is fine.
If memory becomes an issue, switch to delta-based undo (store only the
changed pieces).

## Testing

### Unit tests

```rust
#[test]
fn insert_at_start() {
    let mut pt = PieceTable::from_str("hello");
    pt.insert(0, "say ");
    assert_eq!(pt.content(), "say hello");
}

#[test]
fn insert_at_end() {
    let mut pt = PieceTable::from_str("hello");
    pt.insert(5, " world");
    assert_eq!(pt.content(), "hello world");
}

#[test]
fn insert_in_middle() {
    let mut pt = PieceTable::from_str("helo");
    pt.insert(3, "l");
    assert_eq!(pt.content(), "hello");
}

#[test]
fn delete_range() {
    let mut pt = PieceTable::from_str("hello world");
    pt.delete(5, 11);
    assert_eq!(pt.content(), "hello");
}

#[test]
fn replace_range() {
    let mut pt = PieceTable::from_str("hello world");
    pt.replace(6, 11, "rust");
    assert_eq!(pt.content(), "hello rust");
}

#[test]
fn undo_redo() {
    let mut pt = PieceTable::from_str("hello");
    pt.insert(5, " world");
    assert_eq!(pt.content(), "hello world");
    pt.undo();
    assert_eq!(pt.content(), "hello");
    pt.redo();
    assert_eq!(pt.content(), "hello world");
}

#[test]
fn line_count() {
    let pt = PieceTable::from_str("line1\nline2\nline3");
    assert_eq!(pt.line_count(), 3);
}

#[test]
fn get_line() {
    let pt = PieceTable::from_str("line1\nline2\nline3");
    assert_eq!(pt.line(0), Some("line1".to_string()));
    assert_eq!(pt.line(1), Some("line2".to_string()));
    assert_eq!(pt.line(2), Some("line3".to_string()));
}

#[test]
fn line_col_to_offset() {
    let pt = PieceTable::from_str("abc\ndef\nghi");
    assert_eq!(pt.line_col_to_offset(0, 0), Some(0));
    assert_eq!(pt.line_col_to_offset(1, 0), Some(4));
    assert_eq!(pt.line_col_to_offset(1, 2), Some(6));
    assert_eq!(pt.line_col_to_offset(2, 0), Some(8));
}

#[test]
fn offset_to_line_col() {
    let pt = PieceTable::from_str("abc\ndef\nghi");
    assert_eq!(pt.offset_to_line_col(0), (0, 0));
    assert_eq!(pt.offset_to_line_col(4), (1, 0));
    assert_eq!(pt.offset_to_line_col(6), (1, 2));
}

#[test]
fn grouped_undo() {
    let mut pt = PieceTable::from_str("hello world");
    pt.begin_group();
    pt.delete(5, 11);
    pt.insert(5, " rust");
    pt.end_group();
    assert_eq!(pt.content(), "hello rust");
    pt.undo();
    assert_eq!(pt.content(), "hello world");
}

#[test]
fn modified_flag() {
    let mut pt = PieceTable::from_str("hello");
    assert!(!pt.is_modified());
    pt.insert(5, "!");
    assert!(pt.is_modified());
    pt.mark_saved();
    assert!(!pt.is_modified());
}

#[test]
fn lsp_changes() {
    let mut pt = PieceTable::from_str("hello\nworld");
    pt.insert(5, " beautiful");
    let changes = pt.take_changes();
    assert_eq!(changes.len(), 1);
    assert_eq!(changes[0].start_line, 0);
    assert_eq!(changes[0].start_col, 5);
    assert_eq!(changes[0].new_text, " beautiful");
}

#[test]
fn empty_buffer() {
    let pt = PieceTable::new();
    assert_eq!(pt.len(), 0);
    assert_eq!(pt.line_count(), 1);
    assert_eq!(pt.line(0), Some(String::new()));
}

#[test]
fn large_file_line_access() {
    let content: String = (0..10000).map(|i| format!("line {i}\n")).collect();
    let pt = PieceTable::from_str(&content);
    assert_eq!(pt.line_count(), 10001); // 10000 lines + trailing empty
    assert_eq!(pt.line(5000), Some("line 5000".to_string()));
}
```
